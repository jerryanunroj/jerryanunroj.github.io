# Jerry Anunrojwong

Academic website built with [Hugo](https://gohugo.io/) and the
[Eureka](https://github.com/wangchucheng/hugo-eureka) theme.

## Local development

Install Hugo Extended, then run:

```sh
hugo server
```

The Eureka theme is pinned as a git submodule in `themes/eureka`.

## Deployment

Pushes to `master` run the GitHub Pages workflow in `.github/workflows/hugo.yml`.
